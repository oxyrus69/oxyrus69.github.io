Tentang Saya

Saya adalah seorang pengembang perangkat lunak yang bersemangat dengan fokus pada pengembangan web full-stack. Saya memiliki pengalaman dalam membangun aplikasi yang kuat, skalabel, dan efisien menggunakan berbagai teknologi.

Filosofi saya dalam coding adalah kesederhanaan, efisiensi, dan keterbacaan. Saya percaya bahwa kode yang baik tidak hanya berfungsi, tetapi juga mudah dipahami dan dipelihara oleh orang lain (termasuk diri saya sendiri di masa depan!). Saya selalu mencari cara untuk belajar teknologi baru dan meningkatkan keterampilan saya.

Ketika tidak sedang coding, saya biasanya membaca buku, bermain game, atau menjelajahi alam.
